AutoHotKey: Make your own shortcuts in Windows

Do you want shortcuts for launch faster your favorite program or website? Look here how to use this tool.

What is AutoHotKey?

AutoHotKey is a program for Windows that mades easier to make shortcuts for launch programs and search websites, this program has a file type that they calls 'AutoHotKey Script', in this file you put on it an specific symbols and commands, below I teach you how to do that with some steps for did that.

Step 1: Go to AutoHotKey Website and install the program

Now, in the website, put your cursor on 'Download' button and click on it. And you will see some options, that you can see in the next image.

Click on 'Download Current Version', and the download starts. When the download ends, go to the File Explorer and search the executable downloaded. Then, select them and click the right button of the mouse. After, run the executable as administrator for install the program. When end the install process, you will see an window of AutoHotkey like the next image.

Only you have to close that window, and go to your desktop, or any directory in the File Explorer, if you are questioning, why? This is for make an script of the program, that the creators named like "AutoHotKey Script"

Step 2: Make an script

Now, in the directory that you want to make the script, right click in any white space, go to New (in spanish, 'Nuevo'), and then, 'AutoHotKey Script', then, give it a name. When the script is created, again, right click on this file, and click on 'Edit script' option, this open the notepad or your favorite code editor. For make the script, has an specific syntax. You can see this in the next image, that is my personal script.

As can you see, the file contains many '#' and '!', why? This has a value, each key has a value, and each value is named like command.

  • # = Windows / Meta key
  • ! = Alt key
  • ^ = Control key
  • + = Shift key

For create shortcuts, you have to write the commands with the next syntax: #!(any key of the abecedary)::. And then, put the action what you want.

  • For Show MessageBox, write: MsgBox, 'The message'
  • For Launch programs, write: Run, "Path of the program"

Keep in mind that if the program directory has any white space, you have to quote the directory.

Step 3: Run the script

For run, only save the changes and double click on the file, and that is All! For go the AutoHotkey website, click on the next link.
