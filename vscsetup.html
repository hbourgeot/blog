My Personal Setup of Visual Studio Code

I use a 'mixed' font called Fira Code iScript, in fact, it is the fusion of two fonts, Fira Code, and Script12. The theme, is 'Winter Is Comming', but, you must make sure that you have the italic font enabled, otherwise, you will not be able to see the two font styles in the editor. Although it goes without saying, you can choose any theme, you don't necessarily have to use the one I have.

How to download this font?

For download the font, go to Fira Code iScript github page and click on green button 'Code', then if you use git, clone the repository, but if not, download the repo packaged into a zip file. When download finish, search otf or ttf files for install they, it's depends in the OS do you use. When you have the font installed. Go to your editor, and go to the configutarion. In the field Editor: Font Family, put the next text:

'Fira Code iScript', monospace

Great! You have Fira Code on your VS Code, but, you have to enable font ligatures, for that, go again to configutarion, and search 'Font Ligatures', you will see an section like this:

Click on settings.json and don't worry, the code you see is your personal Visual Studio Code settings, if the option 'editor.fontLigatures' has a value of false, set its value to true, if it has the latter, don't change anything and restart the code editor.

Restart the editor and you will see your new beautiful font!. But, if you don't see the font yet, install a theme in which the title don't contain 'No italics'.
